This layout uses CSS template areas to create a classic website structure. The grid area are defined as:

  • header header header - spans all 3 columns
  • side main main - sidebar takes 1 column, main takes 2
  • footer footer footer - spans all 3 columns

This approach makes layouts incredibly intuitive and easy to modify!
